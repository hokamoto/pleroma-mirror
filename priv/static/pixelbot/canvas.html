Fediverse PixelBot Canvas

This is a proof-of-concept of an interactive canvas. You can add pixels by posting a message to @pixelbot@pynq.limited.systems.

The message is a list of pixels with the following syntax:

 <X-coord><space or comma><Y-coord><space or comma><colour><newline>

The colour is one of 8 colours white, red, green, blue, cyan, magenta, yellow, black, or you can use the single-character codes WRGBCMYK or digits 0-7.

Examples:

            1 1 2
            2,3,red
            4,11,Y
